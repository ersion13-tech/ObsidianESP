# ObsidianESP

**ObsidianESP** is a lightweight utility mod for **Minecraft 1.21.4 (Fabric)**. It highlights nearby Obsidian and Crying Obsidian blocks, allowing you to see them through walls. This tool is indispensable for locating hidden bases, traps, and stashes on anarchy and survival servers.

> [!NOTE]
> This mod is based on the high-performance rendering system from the **Wurst Client**.

---

## Installation

> [!IMPORTANT]
> Always ensure that your **Fabric Loader** and mod version match your Minecraft version (**1.21.4**). Using mismatched versions will cause the game to crash.

1. **Install Fabric Loader** for 1.21.4.
2. **Add Fabric API** to your `mods` folder.
3. **Add ObsidianESP** to your `mods` folder.
4. **(Optional) Install ModMenu** to customize settings directly in-game.

---

## Usage

Once installed, ObsidianESP will automatically start highlighting nearby obsidian blocks. If you have **ModMenu** installed, you can access the configuration GUI to:

Independently enable or disable highlights for **Obsidian** and **Crying Obsidian**.

You can set a custom toggle key in `Options > Controls > Key Binds` (unbound by default).
